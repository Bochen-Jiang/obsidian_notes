# 5.1.4 尾置返回类型
在函数定义的阶段，C++11允许根据某个表达式的结果推断变量的类型
(虽然像脱裤子放屁,至少从书上的例子来看是的)
Ex：
```c++
decltype(5 + 7) add(int a, int b) {
return a + b;
}
```

**但是你不能把 5 + 7 写成 a + b**
**因为编译器没有见过(十分正常)**

但是，我们又提出了一种更为nb的方法————尾置返回类型

```c++
auto add(int a, int b)  -> dectype(a + b) {
 return a + b;
} 
```
