
# 问题一：怎么处理好学习任务和自己的事情的关系


#### 1.一定要意识到，大学和中学不同，很多课程都需要“泛学”，而不是精学

- 大学的知识密度可能比整个高中的知识密度都大
- 不要想像高中一样，全部掌握，而且随着年级提高，









#### 3. 因此，大学的学习任务，一定要根据自身的实际需要，分优先级，而不是面面俱到




#### 4. 不要“分”本位，而是“就业”本位

- “分”本位：以高分为最高优先级，以分数为导向
- “就业”本位：以就业为最高优先级，以就业为最高目标

#### 类型2：能不挂科并且留有余力，目前还不确定未来就业方向

- 一方面，想办法用更短的时间不要挂科
- 剩下的时间分成两半：
	- 一半可以投资到核心基础课程，数学、本专业和英语
	- 剩余**一半时间（更多）**用于探索自己的方向
- 这只是下限，在确定之后，**尤其是确定自己不要读研之后**，甚至需要拿出 **80%** 以上的时间用于自己通用能力的提升


#### 类型3：已经初步明确未来就业方向

- 先花时间搞清楚，基于未来就业方向的实际校招要求
- 哪些课程是影响关键能力积累的？大学的课是不是有意义还是需要**自己自学**







# 总结

## 1.最重要也是最难的，就在于，一定要真正的破除“分”本位的心魔

- 太多人受中学时代的影响，完全是这个追求分数的样子
- 周围的同学大部分依旧是疯狂追求分数
- 要有勇气去走出“大流”，要敢于为自己做的决定负责，而不是跟着大家一起**温水煮青蛙！**


## 2. 如何确定具体优先级，关键是尽早确定未来的就业方向

- 什么学习任务因人而异，不能一概而论
- 关键是看自己的工作类型，是不是需要自己有选择性投入
